# OvTemplates

Collection of custom subscription page templates for OvPanel.

## Installation

Choose your preferred template style below and run the corresponding command on your server to install it. This will replace your current `subscription.html` file.

### Template 1 (Classic Blue)
```bash
curl -L -o /opt/ov-panel/frontend/templates/subscription.html https://raw.githubusercontent.com/erfjab/OvTemplates/master/one.html
```

### Template 2 (Modern Green)
```bash
curl -L -o /opt/ov-panel/frontend/templates/subscription.html https://raw.githubusercontent.com/erfjab/OvTemplates/master/two.html
```

### Template 3 (Minimal Shadcn-like)
**Recommended** - Supports Dark/Light mode.
```bash
curl -L -o /opt/ov-panel/frontend/templates/subscription.html https://raw.githubusercontent.com/erfjab/OvTemplates/master/three.html
```
